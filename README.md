# Apk Re-Signer
A desktop tool for re-sign the .apk files generated by Visual Studio 2026 18.7.1 that might leads to -124 failure in installation on android system.

## Download
https://github.com/Blatke/Apk-Re-Signer/releases
## Requirements for Installation
1. Windows 7.0+.
2. .NET 10.0+
## Requirements for Processing
1. Java.
2. Android Studio.
3. 7-Zip.
4. Your .apk file generated by Visual Studio.
5. Your keystore set by Visual Studio.
